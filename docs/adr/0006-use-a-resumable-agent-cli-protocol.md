# Use a resumable Agent CLI protocol

The official Skill uses an explicit CLI Agent mode as its stable execution contract. Agent mode is non-interactive, suppresses transient progress, and emits one versioned JSON outcome envelope per process invocation while preserving the existing human-readable and field-selected JSON interfaces. A local, unauthenticated, network-free capabilities command advertises integer protocol versions independently of CLI semantic version, and each operational invocation explicitly selects one advertised version. An operation that needs a person returns a structured `action_required` outcome instead of prompting; browser authorization is a resumable two-stage flow so an agent can yield to the user and continue without holding one CLI process open.

The Server remains authoritative for domain state, validation evidence, retry facts, allowed actions, and request identifiers. The CLI preserves those facts and maps them into command-aware next actions; the Skill adds only user intent and local workspace context. This protocol was chosen over composing command-specific flags, parsing display text, or streaming a long-running authorization process because the official Skill needs a portable, restartable contract across agent runtimes. The existing [GitHub CLI interface conventions](0003-follow-github-cli-interface-conventions.md) remain the default for humans and scripts.

## Consequences

Agent-mode schema versions and authentication continuations become compatibility surfaces. Authentication is the only resumable operation in protocol version 1; its continuation never stores or replays a business command. The CLI and, where necessary, the HTTP contract must expose enough structured evidence for completed, in-progress, partial, action-required, failed, indeterminate, and cancelled outcomes. The Skill must not reconstruct missing Server facts, parse human output as a fallback, or silently widen the user's requested operation, and it cannot switch to Agent mode until its complete advertised command surface is covered.
