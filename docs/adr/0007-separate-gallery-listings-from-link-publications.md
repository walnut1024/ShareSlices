# Separate Gallery listings from link Publications

Gallery uses an independent listing aggregate that authorizes a fixed ready Version instead of reusing the Artifact's link Publication. This prevents hidden cross-channel side effects and silent community-content changes, at the cost of a separate authorization, proposal, and serving path. The [product contract](../../PRODUCT.md) owns lifecycle and URL behavior, and the [glossary](../../CONTEXT.md) owns the accepted channel and listing terms.
